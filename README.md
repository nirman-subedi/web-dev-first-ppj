# My Portfolio Website

A simple, responsive multi-page website portfolio featuring Home, Blog, About, and Contact pages.

## Features

- **Home Page**: Hero section with featured content cards linking to other pages
- **Blog Page**: Sample blog posts with dates and descriptions
- **About Page**: Personal information, skills showcase, and journey narrative
- **Contact Page**: Contact information and contact form

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- Vanilla JavaScript (minimal for form handling)

## Pages

1. **index.html** - Home page with hero section and featured content
2. **blog.html** - Blog posts page with article previews
3. **about.html** - About page with skills and personal information
4. **contact.html** - Contact page with contact form and information
5. **styles.css** - Stylesheet with responsive design for all screen sizes

## Design Features

- Responsive grid layout that works on mobile, tablet, and desktop
- Modern navigation bar with active page highlighting
- Consistent color scheme throughout
- Smooth hover effects and transitions
- Mobile-friendly interface

## Color Scheme

- Primary Color: #3498db (Blue)
- Secondary Color: #2c3e50 (Dark Blue)
- Light Background: #ecf0f1 (Light Gray)
- Text Color: #333 (Dark Gray)

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Navigate through the different pages using the navigation bar
4. Customize the content, colors, and information as needed

## Customization

- Update the text content in each HTML file to match your information
- Modify the color variables in `styles.css` `:root` selector to change the theme
- Add more blog posts by duplicating the `.post` structure
- Update contact information with your actual details

## Responsive Breakpoints

- Desktop: 1024px and above
- Tablet: 768px - 1023px
- Mobile: Below 768px
